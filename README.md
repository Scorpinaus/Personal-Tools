# Personal-Tools

This github repo is a list of small, personal tools which I am using on a daily basis:

## Tools

- **codex monitor** - Local Codex usage dashboard for tracking rate-limit status, rolling token usage, conversation token totals, estimated model costs, and usage history from Codex session files. Includes a browser dashboard plus PowerShell and .NET implementations.

- **HTML PDF** - Lightweight local PDF reader built around PDF.js. Supports opening local PDFs, page navigation, search, zoom, fit-to-width, two-page view, document outlines, saved bookmarks, and page text/searchability status.

- **imgToPose** - Batch image-to-pose tool that detects people in images with MediaPipe Pose and renders clean stick-figure/wireframe outputs. Supports front, side, back, or all-view exports, multiple model quality levels, a local web UI, CLI usage, and a local API.

- **poseToAnimation** - Converts an ordered set of pose images into a stickman animation. Detects poses across frames, interpolates movement, and exports MP4 and GIF animations with configurable duration, repeat count, FPS, canvas size, and model quality.

- **vidFrame extractor** - CPU-first local web app for extracting image frames from video files. Supports extracting every frame or frames at fixed second, millisecond, or frame intervals, with JPG/PNG output and a FastAPI backend for local automation.

- **jsonl-viewer.html** - Standalone browser-based JSONL/NDJSON viewer for inspecting local `.jsonl`, `.json`, `.log`, and `.txt` files. Includes multi-file and folder loading, table/JSON/error views, search, filters, sortable columns, selectable fields, record detail inspection, copy, and filtered export.
